pyegt 0.1.0

https://github.com/iannesbitt/pyegt
Ellipsoid-relative geoid and tidal model height lookup
by Ian Nesbitt, NCEAS-UCSB

pyegt is an open source program developed by NCEAS to look up the geoid, tidal, or geopotential model height above the ellipsoid in order to convert model-referenced heights to ellipsoid height (i.e. compatible with Cesium) and vice-versa. For more information, see the About pyegt section.
